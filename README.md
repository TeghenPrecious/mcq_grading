# ExamFlow

An **automated MCQ grading system** built with Node.js, Express, SQLite, and Vanilla HTML/CSS/JS.

## Features

- **Authentication** — Login, signup, logout with JWT
- **Admin Panel** — Create tests, add questions, view grades, export PDF
- 🎓 **Student Portal** — Take timed exams, track progress, view feedback
- **Shuffled Questions** — Unique question/answer order per student
- **Countdown Timer** — Optional per-test time limits
- **PDF Export** — Export all grades to a PDF report
- **One Attempt** — No retakes; browser-close resumes in-progress exam

## Quick Start

```bash
# 1. Clone and install
npm install

# 2. Create environment file
cp .env.example .env
# Edit .env and set your JWT_SECRET and ADMIN_SECRET

# 3. Run in development
npm run dev

# 4. Open http://localhost:3000
```

## Environment Variables

| Variable       | Description                        | Example                       |
| -------------- | ---------------------------------- | ----------------------------- |
| `JWT_SECRET`   | Secret for signing JWT tokens      | `my-super-secret-key`         |
| `ADMIN_SECRET` | Code required to register as admin | `examflow-admin-2024`         |
| `PORT`         | Server port (auto-set on Render)   | `3000`                        |
| `NODE_ENV`     | Environment mode                   | `development` or `production` |

## Default Roles

- **Admin**: Sign up with the `ADMIN_SECRET` code (max 2 admins)
- **Student**: Sign up without a code

## Deploy to Render.com

1. Push this project to a GitHub repository
2. Go to [render.com](https://render.com) → New → Web Service
3. Connect your repo
4. Set environment variables: `ADMIN_SECRET` (choose your own)
5. `JWT_SECRET` is auto-generated by Render
6. Deploy! The `render.yaml` handles everything else.

## Tech Stack

- **Backend**: Node.js, Express.js
- **Database**: SQLite via `better-sqlite3`
- **Auth**: JWT + bcrypt
- **PDF**: pdfkit
- **Frontend**: Vanilla HTML, CSS, JavaScript
